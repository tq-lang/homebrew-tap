# tq Homebrew Tap

Homebrew tap for [`tq`](https://github.com/tq-lang/tq), a command-line TOON/JSON processor.

## Status

Initial setup. Formula will be published after the first tagged `tq` release.

## Install (once formula is published)

```bash
brew tap tq-lang/tap
brew install tq
```

## Upgrade

```bash
brew update
brew upgrade tq
```

## Uninstall

```bash
brew uninstall tq
```
